Save The Spire Savefile Editor
##############################

Python/wxPython based.

Getting Started
---------------
pip install -U wxPython
python3 ./editor.py

The Good:
---------
Opening .autosave files works
Parsing STS .autosave files works
Displaying widgets for changing "settings" values works
